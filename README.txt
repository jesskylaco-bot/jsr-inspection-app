Drop your brand files into this folder:

  logo.png      → shown in the header (max height 100px). PNG with a
                  transparent background looks best. If this file is
                  missing, the app shows a "Company Logo" text fallback —
                  it will not break.

  favicon.ico   → the small browser-tab icon. Optional; if missing the
                  browser just uses its default and logs a harmless 404.

Recommended logo width: ~400–600px (it scales down cleanly).
